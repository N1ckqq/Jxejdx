## ✨ Добавленные функции

- **Оффлайн-вход** — полноценный вход через локальный аккаунт без Microsoft
- **Пакетное скачивание модов** — выделяйте несколько модов галочками и скачивайте за раз
- **Полное меню входа** — доступ к Microsoft, оффлайн и сторонним серверам авторизации

## 🎮 Как использовать оффлайн-вход

1. Откройте экран "Учётная запись"
2. Нажмите "Добавить аккаунт"
3. Выберите "Оффлайн" (离线账号)
4. Введите имя пользователя
5. Выберите созданный аккаунт в списке

## 🔧 Пакетное скачивание модов

1. Перейдите в раздел "Скачать" → "Моды"
2. **Долгое нажатие** на любой мод активирует режим выбора
3. Отмечайте нужные моды галочками
4. Используйте кнопки "Выбрать все" / "Снять выделение"
5. Нажмите плавающую кнопку для скачивания выбранных модов

## 🛠 Сборка

```bash
git clone https://github.com/N1ckqq/Jxejdx.git
cd Jxejdx
./gradlew assembleDebug
```

**Требования:** Android SDK (API 36), NDK 25.2, JDK 17
